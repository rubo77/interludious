# Play Store Assets Instructions

## Required Assets for Play Store

### 1. App Icon
- **Size**: 512 x 512 pixels
- **Format**: PNG (no transparency)
- **Purpose**: Main app icon displayed in Play Store
- **Design**: Use the game logo or a spaceship icon
- **Recommendation**: Use the existing logo from `dev/ideas/logo2.png` as a base

### 2. Feature Graphic
- **Size**: 1024 x 500 pixels
- **Format**: PNG or JPG
- **Purpose**: Featured in the Play Store listing
- **Design**: Show gameplay screenshot or promotional art
- **Recommendation**: Create a composite image showing the ship, pod, and level elements

### 3. Screenshots (Minimum 2, Maximum 8)
- **Size**: At least 1080 x 1920 pixels (phone), 1920 x 1080 pixels (tablet)
- **Format**: PNG or JPG
- **Purpose**: Show gameplay and features
- **Recommended Screenshots**:
  1. Main menu with level selection
  2. Gameplay showing ship and pod
  3. Tractor beam in action
  4. Shooting at bunkers
  5. Level complete screen
  6. Touch controls on mobile

### 4. Promo Video (Optional)
- **Format**: YouTube URL
- **Duration**: 15-30 seconds
- **Purpose**: Show gameplay and features
- **Recommendation**: Record gameplay footage showing level completion

### 5. Banner Image (Android TV)
- **Size**: 1280 x 720 pixels
- **Format**: PNG or JPG
- **Purpose**: Featured on Android TV
- **Recommendation**: Only needed if targeting Android TV

## Asset Creation Guidelines

### Color Scheme
- Primary: Cyan (#00ffff) - Ship color
- Secondary: Green (#00ff88) - Success/completed
- Accent: Orange (#ff6600) - Explosions
- Background: Dark (#000000) - Space theme

### Typography
- Font: Arial or similar sans-serif
- Style: Clean, modern, readable
- Contrast: High contrast for readability

### Branding
- Include app name "Interludious" in feature graphic
- Use consistent styling across all assets
- Maintain space/sci-fi theme

## Asset Sources

### Existing Assets
- Logo: `dev/ideas/logo2.png`
- Background: `dev/ideas/*.png` (background images)
- Game sprites: In-game assets from the game

### Screenshot Capture
To capture screenshots from the game:
1. Run the game in browser or mobile
2. Use screenshot tools (Android: Power + Volume Down)
3. Crop to required dimensions
4. Add text overlays if needed

### Image Editing Tools
- GIMP (free, open-source)
- Photoshop (commercial)
- Canva (online, free tier available)
- Figma (online, free tier available)

## Asset Validation

### Before Upload
- Check dimensions match requirements
- Verify file format (PNG/JPG)
- Ensure no transparency in app icon
- Test on different screen sizes
- Check contrast and readability

### After Upload
- Preview in Play Store Console
- Test on actual devices
- Verify appearance in search results
- Check featured graphic placement

## Upload Instructions

### Play Store Console
1. Go to Google Play Console
2. Select your app
3. Navigate to "Main Store Listing"
4. Upload each asset to its respective section
5. Save and publish changes

### Version Notes
- Update assets with each major version
- Keep backup of original assets
- Document asset changes in changelog
